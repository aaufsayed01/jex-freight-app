# JEX Freight App 

A full-stack logistics and freight management system designed to manage quotes, pricing, bookings, shipments, and documents.

## Tech Stack
### Backend
- Node.js
- Express
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT Authentication
- AWS S3
- Swagger API Docs

### Frontend
- React
- TypeScript
- (UI & flows in progress)

## Features
- Role-based authentication (Admin, Ops, Corporate Client)
- Quote request & pricing workflows (AIR / SEA)
- Booking & shipment management
- Document upload & secure downloads
- Audit logs & notifications
- Dockerized local setup
